Objetivo do Exercício: Este exercício tem como objetivo fornecer uma introdução prática aos conceitos básicos de HTML (Hypertext Markup Language) e CSS (Cascading Style Sheets). Por meio deste exemplo, você terá a oportunidade de aprender como estruturar uma página web e estilizá-la para criar uma experiência visualmente atraente para os usuários.

Conteúdo da Página: A página de exemplo incluirá elementos comuns encontrados em muitos sites, como um cabeçalho, um menu de navegação, seções de conteúdo e um rodapé. Cada elemento será marcado adequadamente com tags HTML e estilizado usando CSS.
